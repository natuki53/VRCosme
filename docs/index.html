VRCosmeは、VRChat向け写真レタッチに特化したWindowsアプリです。 読み込み、補正、比較、書き出しまでを1画面で完結できます。

⬇最新版をダウンロード リリース一覧 GitHub
対応OS Windows 10 / 11
アーキテクチャ x64 (64bit)
入力形式 PNG / JPG / BMP / TIFF / WEBP
出力形式 PNG / JPEG（品質指定）

主な機能

実装済みの主要機能を用途別に整理しています。

Base基本補正

明るさ、コントラスト、ガンマ、露出、彩度、色温度、色かぶりを調整。

Fine詳細補正

シャドウ、ハイライト、明瞭度、シャープ、周辺減光で質感を仕上げ。

Crop構図調整

アスペクト比トリミング、左右・上下反転、90度回転に対応。

Compare比較表示

補正後、補正前、分割比較を切り替え。分割ラインはドラッグ可能。

Assist作業支援

Undo/Redo、三分割グリッド、ルーラー、ドラッグ&ドロップ読み込み。

Export書き出し

PNG/JPEGへ出力。JPEGは品質を1〜100で指定できます。

解説セクション（使い方）

初回利用時の流れに沿って、基本操作を8ステップでまとめています。

1画像を読み込む

  • ファイル > 画像を開く または Ctrl + O
  • プレビューへドラッグ&ドロップでも読み込み可能
  • 最近使った画像 から再オープン可能

2基本補正を行う

  • 明るさ、コントラスト、露出、ガンマで全体を整える
  • 彩度、色温度、色かぶりで色味を調整
  • 各項目のリセットボタンで個別初期化

3詳細補正で仕上げる

  • シャドウ/ハイライトで階調を調整
  • 明瞭度、シャープで輪郭の見え方を調整
  • 周辺減光で視線誘導を作る

4トリミングと構図調整

  • 1:1、16:9、3:4などの比率を選択
  • トリミング枠をドラッグして位置とサイズを調整
  • 90度回転、左右/上下反転に対応

5補正前後を比較

  • 補正後 と 補正前 を切り替え
  • 分割 モードで左右同時比較
  • 中央ラインをドラッグして比較位置を変更

6ズームとパン

  • マウスホイールでズーム
  • 左ドラッグまたは中ドラッグでパン移動
  • 下部ツールバーの ＋ / − でも操作可能

7書き出し

  • 書き出し または Ctrl + Shift + S
  • PNG/JPEGを選択
  • JPEGは品質（1〜100）を指定

8設定を保存する

  • 設定 > アプリ設定... で環境を最適化
  • テーマ、初期保存先、初期形式、初期品質を設定
  • グリッド表示もここで変更可能

ショートカットと補足

よく使うショートカット

  • 画像を開くCtrl + O
  • 書き出しCtrl + Shift + S
  • 元に戻すCtrl + Z
  • やり直すCtrl + Y
  • やり直す（代替）Ctrl + Shift + Z

トラブル時

  • ヘルプ > ログの書き出し... でログをZIP保存
  • 不具合報告時はログ添付で調査が速くなります
